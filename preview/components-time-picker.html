Time Picker

Scrollable hour and minute columns with infinite drum-roll scroll. Supports 12- and 24-hour formats, configurable minute steps, manual hour entry, and AM/PM toggle. Follows WAI-ARIA dialog pattern.

Default — 12-Hour, 1-Minute Step

Spec format="12" · minuteStep={1} (default) · infinite drum-roll scroll on both columns
Panel: --parchment-200 bg · 1.5px solid --ink-brown border · 4px 4px 0 --parchment-400 shadow
Header: --parchment-300 bg · click hour digits to type manually
Selected item: --blood bg · --parchment-100 text · all occurrences across infinite list highlighted

Pre-filled Value — "09:37"

Pre-filled value="09:37" — initialises internal state; infinite scroll columns jump to selected item on open
Scroll uses scrollTop (no scrollIntoView) to avoid breaking embedded previews

24-Hour Format

24h mode format="24" · no AM/PM column · hour column shows 00–23
Header label reads "Hour (24)" · display shows "14:30" style

Format Comparison

Minute Step Variants

minuteStep prop 1 → 60 rows (default) · 5 → 12 rows · 10 → 6 rows · 15 → 4 rows · 30 → 2 rows
All variants use infinite drum-roll scroll

Disabled State

On Card Surface

Accessibility

ARIA pattern Trigger: <button aria-haspopup="dialog" aria-expanded>
Panel: role="dialog" aria-label="Time picker" aria-modal="false"
Hour col: role="listbox" aria-label="Select hour"
Minute col: role="listbox" aria-label="Select minute"
AM/PM col: role="listbox" aria-label="AM or PM"
Each option: role="option" aria-selected
Header: aria-hidden="true" (decorative, state described by options)
Escape key: closes panel and returns focus to trigger
Click outside: closes panel
:focus-visible only — gold ring (#7a5a22) on light; parchment ring on Confirm btn

Component API

Props — TimePicker.jsx value — "HH:MM" 24h string | null · initial committed value
onChange — (val: "HH:MM" | null) => void · fires on Confirm / Clear
format — '12' | '24' · default '12'
minuteStep — 1|5|10|15|30 · default 1
disabled — boolean · default false
placeholder — string · default 'Select the hour…'
ariaLabel — string · names trigger + dialog (default 'Time picker')
id — string · sets id on trigger for <label for> association
