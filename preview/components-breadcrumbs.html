Breadcrumbs

Wayfinding trail for deep content hierarchies. Display-face links in faded ink, gold separators, iron-gall current page. Follows the WAI-ARIA breadcrumb navigation pattern.

Standard — Three Levels

Spec <nav aria-label="Breadcrumb"> → <ol> → <li>
.breadcrumb__link: --font-display · 0.8125rem · ls 0.05em · --ink-faded
:hover → color --gold-deep · border-bottom 1px solid --gold-deep
.breadcrumb__sep: "›" · aria-hidden · --gold · 0.75rem
.breadcrumb__current: --ink-black · no href · aria-current="page"

With Location Eyebrow

Eyebrow pairing eyebrow prop wraps .eyebrow + <nav> in a column-flex group with gap: --space-1
.eyebrow: --font-display · 13px · ls 0.3em · uppercase · --gold-deep
Recommended for spell/creature/item detail pages

Page Header — With Divider

Collapsed — Deep Hierarchy

Collapsed variant maxVisible={3} prop collapses middle levels to an ellipsis "…"
aria-label="more pages" on the ellipsis <span>
Rule: always shows root + ellipsis + last (maxVisible−2) ancestors + current
Collapse when 3 or more intermediate levels are hidden

On Card Surface (Parchment-200)

Card surface Default token colours hold on --parchment-200 card backgrounds.
--ink-faded links: ~4.8:1 on --parchment-200 (passes AA for UI components)
--gold-deep hover: ~4.1:1 on --parchment-200 (passes AA large text)

Dark Surface (Sidebar / Dark Panel)

Dark modifier (dark prop) link: rgba(168,141,90,0.65) → hover: --gold-bright (#d4a548)
sep: --gold-deep · current: --parchment-100
:focus-visible outline: 2px solid --parchment-100
Use on --ink-black / --shadow panel backgrounds only

Accessibility

ARIA pattern (WAI-ARIA 1.2) <nav aria-label="Breadcrumb"> — named landmark
<ol> — ordered list; reading order conveys path depth
aria-current="page" — on the final, non-linked item
aria-hidden="true" — on every separator glyph
aria-label="more pages" — on the ellipsis span
:focus-visible only — never suppress outline without replacement
Light surface: outline 2px solid --gold-deep, offset 2px
Dark surface: outline 2px solid --parchment-100, offset 2px
No scrollIntoView() usage

Component API

Props — Breadcrumb.jsx items — { label: string, href?: string }[] (required)
           Last item without href becomes aria-current="page"
eyebrow — string | null · renders eyebrow label above trail
dark — boolean · flips colours for dark panel backgrounds
maxVisible — number | null · collapses middle levels to ellipsis
ariaLabel — string · names the <nav> landmark (default: "Breadcrumb")

Token Reference

Tokens used font: --font-display (links + current) · --font-body (ellipsis)
size: --text-label (0.8125rem / 13px)
spacing: letter-spacing 0.05em
link colour: --ink-faded (#6b4f35)
link hover: --gold-deep (#7a5a22)
separator: --gold (#b8893b) · "›" glyph
current: --ink-black (#1a1410)
ellipsis: --ink-faded · Roboto body face
focus ring: 2px solid --gold-deep · offset 2px
transition: color 0.12s ease · border-bottom-color 0.12s ease
