Inline Code

Use --blood for primary actions. Apply border-radius: 1px maximum. The --font-mono stack uses JetBrains Mono as the display typeface for all token names and hex values.

Syntax Token Classes

.tok-selector — keys, selectors (gold-bright)
.tok-property — property names (cool blue)
.tok-value / .tok-string — values, strings (sage)
.tok-comment — comments, italic (ink-faded)
.tok-keyword — keywords, booleans (muted red)
.tok-number — numeric literals (warm tan)
