## Quick Reference

- **Display font:** Tiamat Condensed SC (files in `fonts/`) with Cinzel as fallback
- **Body font:** Roboto (Google Fonts)
- **Flavor font:** IM Fell English italic (Google Fonts)
- **Mono font:** JetBrains Mono (Google Fonts)
- **Primary bg:** `#f5ecd7` (parchment-100)
- **Primary action:** `#7a1212` (blood)
- **Accent:** `#b8893b` (gold)
- **All tokens:** `colors_and_type.css`
- **No border-radius > 2px. No blur shadows. No emoji. No Emerald + Sapphire together.**
- **UI kit:** `ui_kits/compendium/index.html`
